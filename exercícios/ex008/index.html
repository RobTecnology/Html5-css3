Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b> (Nao semantica, ou seja, sem significado)

Ja nesta frase temos um termo em destaque usando a tag <strong> (semantica, com significado)

Itálico/Enfase

Nesta frase temos um termo em Itálico usando a tag <i>. (não semantica)

Já nesta frase temos um termo em enfase usando a tag <em>. (semantica)

Texto marcado

Podemos marcar um texto como esse daqui usando a tag <mark>.

Este texto também pode ser marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno.

Texto deletado

Podemos marcar um texto como excluído para que ele possa ser lido mas, não considerado. Usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois. Usando a tag <ins>

Existe também o sublinhado com a tag <u>. (não semantica)

Texto sobrescrito

Um exemplo é a equação x 20 + 45 = 10. Usando a tag <sup>.

Texto subscrito

Um exemplo é a formula H2O. Usando a tag <sub>.
